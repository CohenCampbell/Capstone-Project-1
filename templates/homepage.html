Welcome

This is a tool used to search a specified Spotify podcast for specific guests, topics, or episodes. The tool works by first selecting a podcast, then using a keyword, it looks through the descriptions of all the episodes for the selected podcast and returns the episode titles with matching results. To start, click the button below and find the podcast you'd like to search through, then once selected, input a keyword and all episodes with matching results will be displayed. The tool can only search through 50 episodes at once. It will look through the most recent episodes and can be offset to check older episodes.
