Welcome

This is a tool used to search a specified Spotify podcast for specific guests, topics, or episodes. The tool works by first searching for and selecting a podcast, then using a keyword, it looks through the descriptions and titles of all the episodes for the selected podcast and returns the matching results. To start, click the button below and find the podcast you'd like to search through, then once selected, input a keyword and all episodes with matching results will be displayed
